simple web is cute

don't ruin it

The internet is at the same time better and worse than it used to be in the '90s.

We don't need huge websites and massive frameworks to convey ideas.

We don't need excessive images, styling and scripts.

We probably don't need libraries for basic stuff that can be done in vanilla JS.

Fancy CSS is still nice, but only if we drop unused code instead of including all of Bootstrap or something.

Static sites are niiiiiice.

There should usually be enough okay fonts on people's devices to just refer to font-family: 'Helvetica Neue', Lato, Noto, Roboto, 'everything but the kitchen sink', sans-serif.

It said should just now. There aren't enough okay fonts on people's devices. We need to find a way to encourage people to install some.
